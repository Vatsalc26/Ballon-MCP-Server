# Support

Balloon MCP is an early public alpha.

## Support Paths

For normal bugs, questions, and contribution-sized problems:

1. use the GitHub Issues tab

For security-sensitive issues:

1. do not open a public issue
2. follow [SECURITY.md](./SECURITY.md)

## Good Issue Types

The best public issues right now are:

1. bug reports
2. benchmark reports
3. host compatibility reports
4. bounded release or listing prep tasks

## Current Expectation

Support is currently open-source alpha support, not a paid support channel or enterprise SLA.
